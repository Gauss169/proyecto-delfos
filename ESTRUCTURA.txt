================================================================================
                            FLUJO DE TRABAJO
================================================================================

1️⃣ DESARROLLO DE MODELOS
   notebooks/model_training.ipynb
   ↓
   Entrenar modelo
   ↓
   Exportar 3 archivos .pkl a pipeline/models/

2️⃣ INTEGRACIÓN
   Actualizar pipeline/config.py
   ↓
   Probar con scripts/test_prediction.py

3️⃣ TESTING LOCAL
   cd pipeline
   python bot.py
   ↓
   Verificar logs en pipeline/logs/trading.log

4️⃣ DEPLOYMENT
   Subir a GitHub (privado)
   ↓
   Conectar a Railway/AWS/GCP/Azure
   ↓
   Configurar variables de entorno
   ↓
   Deploy automático ✅

================================================================================
                          COMANDOS PRINCIPALES
================================================================================

# Setup inicial
python scripts/setup.py

# Probar predicción
python scripts/test_prediction.py

# Ejecutar bot localmente
cd pipeline
python bot.py

# Copiar modelos
python scripts/copy_models.py

# Iniciar Jupyter
cd notebooks
jupyter notebook

# Docker local
cd deployment
docker-compose up -d

# Ver logs Docker
docker-compose logs -f trading-bot
